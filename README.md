# voohu-hardware-libraries
PCB footprints, schematic symbols, and 3D STEP models for VOOHU network transformers and connectors
# VOOHU PCB Libraries for Ethernet Hardware Design

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![KiCad](https://img.shields.io/badge/KiCad-8.0+-green.svg)](https://www.kicad.org/)

VOOHU provides PCB footprints and 3D models for our network transformers, RJ45 connectors, and SFP cages. These libraries are designed for industrial Ethernet applications including PLCs, industrial control, and smart security.

## 📦 What's Included

### Network Transformers
- VT-100 Series (10/100Mbps) - SMD and DIP packages
- VT-1G Series (1 Gigabit) - SMD and DIP packages
- VT-POE Series (802.3af/at/bt) - SMD packages

### RJ45 Connectors
- 1x1 vertical and horizontal mount
- 1xN multi-port configurations
- With and without integrated magnetics
- With LED indicators

### SFP Cages
- Single and stacked configurations
- With EMI spring fingers

## 🚀 Quick Start

### KiCad
1. Clone this repository
2. In KiCad: Preferences → Manage Symbol Libraries
3. Add `/symbols` folder
4. Repeat for footprints in `/footprints`

### Altium
1. Download as ZIP
2. Extract and add `.SchLib` / `.PcbLib` files to your project
